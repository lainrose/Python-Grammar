## 04. 튜플 자료형

### 튜플의 생성 방법
- - -
- 리스트는 [과 ]으로 둘러싸지만 튜플은 (와 )으로 둘러싼다.
- 리스트는 그 값의 생성, 삭제, 수정이 가능하지만 튜플은 그 값을 바꿀 수 없다.
```python
>>> t1 = ()
>>> t2 = (1,)
>>> t3 = (1, 2, 3)
>>> t4 = 1, 2, 3
>>> t5 = ('a', 'b', ('ab', 'cd'))
```
- t2 = (1,)처럼 단지 1개의 요소만을 가질 때는 요소 뒤에 콤마(,)를 반드시 붙여야 한다.
- t4 = 1, 2, 3처럼 괄호()를 생략해도 된다.

### 튜플의 인덱싱과 슬라이싱, 더하기(+)와 곱하기(*)
- - -
#### 1. 인덱싱하기
```python
>>> t1 = (1, 2, 'a', 'b')
>>> t1[0]
1
>>> t1[3]
'b'
```
#### 2. 슬라이싱하기
```python
>>> t1 = (1, 2, 'a', 'b')
>>> t1[1:]
(2, 'a', 'b')
```
#### 3. 튜플 더하기
```python
>>> t2 = (3, 4)
>>> t1 + t2
(1, 2, 'a', 'b', 3, 4)
```
#### 4. 튜플 곱하기
```python
>>> t2 * 3
(3, 4, 3, 4, 3, 4)
튜플의 곱하기(반복) 예를 보여 준다.
```
